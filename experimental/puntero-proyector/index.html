Puntero de Proyector

Prueba si el lapiz/dedo de un proyector interactivo de tiro ultracorto (Epson BrightLink, BenQ, etc.) llega al navegador como un puntero utilizable, y con que precision y datos.

Dibuja en la pantalla completa; el panel lateral y la bitacora muestran cada evento en vivo.

Requisitos para que funcione

  • El proyector debe estar conectado por USB ademas del cable de video: por ahi viaja el reporte HID del lapiz/dedo.
  • Debe estar calibrado con el software oficial del fabricante para el area exacta de proyeccion.
  • El sistema operativo debe reconocerlo como dispositivo de puntero (mouse, touch o pen) — se ve en Dispositivos de Windows/macOS.
  • pointerType puede llegar como "pen", "touch" o incluso "mouse" segun el driver: el codigo debe aceptar cualquiera.
  • La pagina necesita touch-action: none en la zona interactiva (ya incluido aqui) para que el navegador no intente hacer scroll/zoom con los trazos.

Usa la rejilla de calibracion para verificar que el trazo cae exactamente donde toca el lapiz en cada esquina y el centro.
