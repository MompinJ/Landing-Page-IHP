Nivel 1

Entrada de datos

Variables en crudo ingresadas al sistema: peso, talla, perímetro de muñeca, pliegues cutáneos PCB/PCT, actividad física.

→

Nivel 2

Motor backend

Automatización instantánea de fórmulas complejas: Harris-Benedict, IOM, Metropolitan Life Insurance, pliegues.

→

Nivel 3

Salida diagnóstica

Diagnósticos en tiempo real, precisos y listos para la intervención: gasto energético total, % de grasa corporal, complexión, diagnóstico IMC.

Infraestructura

Stack tecnológico de grado empresarial

Ecosistema unificado bajo Full Stack TypeScript para reducir errores y asegurar coherencia total.

EL CLIENTE

Azure Static Web Apps ejecutando React + Vite.

Tiempos de carga ultrarrápidos, alta disponibilidad global y estabilidad ante picos de uso.

EL SERVIDOR

API REST escalable con Node.js y NestJS.

Arquitectura modular, seguridad robusta y validación estricta mediante DTOs.

PERSISTENCIA

PostgreSQL

Integridad relacional inquebrantable para historiales médicos sensibles.
